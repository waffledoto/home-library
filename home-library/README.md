# ☕ Домашняя Библиотека

Веб-приложение для управления личной коллекцией книг с встроенным PDF-ридером. Разработано с использованием Next.js, TypeScript, PostgreSQL и Tailwind CSS.

## 🎨 Дизайн

Приложение выполнено в красивых кофейных тонах с градиентами и современным UI:
- **3 колонки** для управления статусом книг: "Не начатые", "Читаю", "Прочитанные"
- **Встроенный PDF-ридер** для чтения книг прямо на сайте
- **Адаптивный дизайн** для работы на любых устройствах
- **Красивые анимации** и плавные переходы

## ✨ Функциональность

### Управление книгами
- ➕ Добавление новых книг с обложкой, описанием и файлом книги
- ✏️ Изменение статуса книги (не начата / читаю / прочитана)
- 📊 Отслеживание прогресса чтения по страницам
- 💾 Сохранение прогресса чтения
- 🗑️ Удаление книг

### PDF-ридер
- 📖 Чтение PDF файлов прямо в браузере
- 📁 Возможность открыть локальный PDF-файл с компьютера
- 🔄 Автоматическое сохранение текущей страницы
- 📱 Удобный интерфейс для навигации по страницам

### Загрузка файлов
- 🖼️ Загрузка обложек книг (JPEG, PNG, WebP, GIF)
- 📚 Загрузка файлов книг (PDF, EPUB, FB2, MOBI, DJVU, TXT)
- 💾 Файлы хранятся в директории `public/uploads`

## 📖 Использование

### Добавление книги

1. Нажмите кнопку **"Добавить книгу"** в шапке
2. Заполните поля:
   - **Название книги** (обязательно)
   - **Автор** (обязательно)
   - **Описание** (необязательно)
   - **Обложка** (необязательно, изображение)
   - **Файл книги** (необязательно, PDF)
3. Нажмите **"Добавить книгу"**

### Чтение книги

1. Кликните на карточку книги
2. Откроется PDF-ридер
3. Если у книги есть PDF-файл, он загрузится автоматически
4. Можно открыть локальный PDF-файл кнопкой **"Открыть PDF"**

### Изменение статуса

1. В PDF-ридере нажмите на кнопку статуса
2. Выберите новый статус:
   - 📚 **Не начата** - книга ещё не начата
   - 📖 **Читаю** - книга читается
   - ✅ **Прочитана** - книга прочитана
3. Нажмите **"Сохранить"** для сохранения прогресса

## 🔌 API Эндпоинты

### Книги
- `GET /api/books` — Получить все книги
- `POST /api/books` — Создать новую книгу
- `PUT /api/books/[id]` — Обновить книгу
- `DELETE /api/books/[id]` — Удалить книгу

### Загрузка файлов
- `POST /api/upload` — Загрузить файл (обложка или книга)
- `DELETE /api/upload?path=/uploads/...` — Удалить файл
- `GET /api/files/books/[filename]` — Скачать/просмотреть файл книги

## 📝 Примечания

- **База данных:** Supabase PostgreSQL (облачная, бесплатная, ~500MB бесплатно)
- **Хранение файлов:** локально в `public/uploads/` (для продакшена — Supabase Storage или Vercel Blob)
- **Данные книг:** сохраняются в базу данных, а НЕ в localStorage
- Приложение поддерживает как загруженные на сервер файлы, так и локальные файлы

## 🐛 Решение проблем

### Ошибка подключения к базе данных
```
Database URL not found. Set POSTGRES_URL...
```
**Решение:** Убедитесь, что файл `.env.local` существует и содержит правильную строку подключения из Supabase.

### Ошибка SSL подключения
**Решение:** Убедитесь, что используете строку подключения в режиме **Transaction** (порт 6543), а не Session (порт 5432).

### Ошибка при запуске
Убедитесь, что все зависимости установлены:
```bash
npm install
```
